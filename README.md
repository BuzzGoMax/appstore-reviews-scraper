# App Store Reviews Scraper

Bulk extract iOS App Store reviews for any app, in any country. Pulls title, body, rating, author, version, and timestamps. App metadata included free.

## Pricing

- $0.05 per actor start
- $0.005 per review extracted

## Use cases

- Mobile app developer competitive intel
- ASO (App Store Optimization) research
- Sentiment analysis on competitor releases
- Tracking review trends across markets and versions
- AI/LLM training on app feedback
