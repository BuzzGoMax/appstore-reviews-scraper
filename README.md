[Appstore Reviews Scraper](https://apify.com/focused_vanguard/appstore-reviews-scraper?fpr=data)

Scrape app reviews from **Apple App Store** and **Google Play Store**.

## Features

- iOS and Android app reviews
- Ratings, review text, dates
- App version tracking
- Multiple country support (App Store)

## Supported URLs

- Apple App Store: `https://apps.apple.com/us/app/instagram/id389801252`
- Google Play: `https://play.google.com/store/apps/details?id=com.instagram.android`

## Use Cases

- **App Developers**: Monitor user feedback
- **Product Managers**: Track feature requests
- **Marketers**: Competitive app analysis

## Input Example

```
{
    "url": "https://apps.apple.com/us/app/instagram/id389801252",
    "maxReviews": 100,
    "country": "us"
}
```
